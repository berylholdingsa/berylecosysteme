# Plan reponse fraude

- Niveau 1: anomaly locale (surveillance continue)
- Niveau 2: pattern confirme (gel partiel)
- Niveau 3: fraude coordonnee (war-room + escalade banque)

- Quarantaine transactions suspectes
- Notification equipe compliance
- Alimentation rapport incident reglementaire

## Actions humaines
- Revue AML/forensic
- Contact partenaire bancaire
- Decision de remise en service progressive
